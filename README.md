# [CE118.Q11.VMTN - Nhóm 1] Đồ án thiết kế vi xử lý 16-bit theo kiến trúc tập lệnh đơn giản
+) GitHub này bao gồm (Sẽ bổ sung sau khi báo sản phẩm final):
- CPU_16bit_Assembler.cpp: Trình biên dịch Assembler viết bằng C++, chuyển đổi mã Assembly từ tệp nguồn 'input.asm' thành mã máy và xuất kết quả dưới định dạng tệp 'program.hex' ở dạng mã thập lục phân.
- Các mạch thiết kế thành phần của CPU 16-bit (.bdf/ .bsf).
- Các tệp mô phỏng dạng sóng kiểm tra các thành phần của CPU 16-bit (.vwf).
- 'program1.hex' chứa các lệnh số học (RRR).
- 'program2.hex' chứa các lệnh thao tác với DMem và Imm (RRI + RI).
- 'program3.hex' chứa các lệnh giả + lệnh rẽ nhánh (beq/ jnz).
- 'program4.hex' chứa chương trình thao tác với I/O Controller.
- Tệp 'clk.sdc' dùng để khởi tạo clk 50.0 MGhz để kiểm thử thông số mạch.
